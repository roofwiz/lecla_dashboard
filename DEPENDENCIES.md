# Project Dependencies

## Frontend (React + Vite)
- **Node.js**: Runtime environment
- **React**: UI Library
- **Vite**: Build tool
- **TailwindCSS**: Styling (Optional, strictly if requested, but using vanilla CSS as per instructions unless needed)
    - *Note: User instructions prefer Vanilla CSS, but Tailwind is common. Will stick to Vanilla CSS unless prompted.*
- **Axios** (or native fetch): For API calls

## Backend (Python)
- **Python 3.x**
- **FastAPI**: Web framework
- **Uvicorn**: ASGI server
- **Requests/HTTPX**: For external API integration
- **Python-dotenv**: Environment variable management

## External APIs
- **Job Nimbus API**: For job management data
- **Company Cam API**: For photo documentation
- **Google API**: (Maps, Calendar, etc. - TBD)

## Tools
- **Git**: Version control
- **VS Code**: Recommended IDE
